About

Solomon Rosenthal is a Los Angeles based composer, multi-instrumentalist, music producer, visual artist, and software designer.

Discography
    Software
    • Euclidian Gate Free

      A gate sequencer built on the Euclidian algorithm, which distributes patterns evenly across a set number of steps. Includes a Rotate parameter and preset storage.

    • MCLFO Free

      A Max for Live modulation device with nine sine wave LFOs, each independently mappable to any parameter in Ableton Live.

    • Chaotic Resonator Free

      A Max for Live audio effect for experimental sound design featuring four delay lines patched into themselves, producing unpredictable resonant feedback textures.

    • Mesh Visualizer Free

      An audio-reactive Max for Live device that uses audio from Ableton Live to modulate a generative mesh lattice, creating real-time visuals from any audio source.

    • Fractal Dream Free

      An Ableton visualizer inspired by the fractal dream strange attractor. Animations are triggered by incoming MIDI notes with four parameters to shape the resulting image.

    • PitchBlur $7.99

      An FFT-based spectral multi-effect Max for Live device. Includes spectral denoise and deharmonic filters alongside spectral blur, gate, and pitch shift.
